Getting started

Install @ascentsparksoftware/angular-calendar, wire the date adapter once, and render your first view in a few lines. The library is standalone, signals-first, zoneless and SSR-safe — you bring your own event data and templates, and it drops onto any color scheme.

Install the package alongside its runtime peers date-fns, date-fns-tz and rrule. The Angular peer dependencies — @angular/core, @angular/common and @angular/platform-browser — must match your Angular major (^22, ^21 or ^20).

Optional secondary entry points keep the core lean; import them only when you need them:

  • @ascentsparksoftware/angular-calendar/date-fns — the date adapter.
  • @ascentsparksoftware/angular-calendar/recurrence — RRULE recurrence.
  • @ascentsparksoftware/angular-calendar/export — CSV / ICS / Excel / print.
